# Grafana – Quick Overview

**What is Grafana?**  
Grafana is an open-source platform for **monitoring and observability**. It allows you to **visualize metrics, logs, and traces** from various data sources, such as Prometheus, Elasticsearch, Loki, and more. You can build dashboards, set up alerts, and explore time-series data interactively.  

**Official documentation:** [https://grafana.com/docs/](https://grafana.com/docs/)

---

## Connecting to Grafana locally

1. **Default access:**  
   After installing Grafana locally, it runs by default on:

```
http://localhost:3000
```

2. **Login:**  
- Default username: `admin`  
- Default password: `admin` (you’ll be prompted to change it on first login)

3. **Create dashboards:**  
- Click **Create → Dashboard → Add new panel**  
- Enter your query or metric to visualize  
- Customize graphs, tables, or other panels as needed

---

## Tips

- Grafana supports **multiple visualization types**, allowing you to monitor different aspects of your applications in one place.  
- Panels are customizable and support alerts when metrics exceed thresholds.  
- You can share dashboards via links or snapshots.
